Mock JSON API

This is a mock API server for testing the JQ Proxy Service.

Available Endpoints:

  • GET /posts.json - List of posts
  • GET /users.json - List of users

Example Proxy Request:

curl -X POST http://localhost:8080/proxy/jsonplaceholder/posts.json \
  -H 'Content-Type: application/json' \
  -d '{
    "method": "GET",
    "body": null,
    "transformation": {
      "posts": "$[*].{id: id, title: title}",
      "count": "$.length"
    }
  }'
